<Section icon={Swords} title="Utilizando o Discord do Evomon para Códigos e Recompensas">
O servidor do Discord do Evomon é o local ideal para os jogadores encontrarem e resgatarem códigos ativos, que muitas vezes concedem recompensas valiosas no jogo, como Moedas, Frutas de EXP ou outros itens úteis. Os desenvolvedores lançam novos códigos frequentemente como parte de marcos, eventos ou engajamento da comunidade.

<Callout type="success" title="Recompensas Instantâneas">
Muitos **códigos do Evomon** têm tempo limitado. Estar no Discord garante que você receba notificações de novos códigos assim que eles forem lançados, maximizando suas recompensas gratuitas.
</Callout>

**Passos para Resgatar Códigos do Discord:**
<Steps>
  <Step num="1" title="Entre no Servidor Oficial">
    Use um link verificado para entrar no servidor oficial do Discord do Evomon. Procure pelo canal "announcements" ou "codes".
  </Step>
  <Step num="2" title="Localize os Códigos Ativos">
    Verifique os canais dedicados #codes ou #announcements para os últimos **códigos do Evomon** ativos. Copie-os exatamente como aparecem.
  </Step>
  <Step num="3" title="Inicie o Evomon no Roblox">
    Inicie o jogo Evomon a partir de sua página oficial no Roblox.
  </Step>
  <Step num="4" title="Acesse o Menu de Configurações">
    Uma vez no jogo, clique no ícone da engrenagem (configurações), geralmente localizado perto do topo da tela.
  </Step>
  <Step num="5" title="Cole e Resgate">
    Cole o código copiado na caixa de entrada e pressione "OK" para resgatar suas recompensas. Certifique-se de que nenhum espaço extra foi colado.
  </Step>
</Steps>

| Status do Código | Descrição                                       | Ação Necessária                                          |
| :---------- | :------------------------------------------------ | :------------------------------------------------------- |
| **Ativo**  | Atualmente funcionando e fornece recompensas.           | Resgate imediatamente para garantir as recompensas.                     |
| **Expirado** | Não funciona mais; as recompensas não podem ser resgatadas.  | Esses códigos geralmente são movidos para uma lista de "códigos expirados". |
| **Próximo**| Anunciado, mas ainda não ativo.                     | Fique de olho no Discord para anúncios de ativação. |

</Section>
